Este script ruby organiza a área de trabalho da seguinte maneira:

Um novo diretório é criado no local em que o arquivo .rb está e o nome do mesmo é definido como desktop-organizer

Os arquivos presentes no diretório em questão são observados e posteriormente novos são criados com a nomenclatura referente a data da ultima edição de cada arquivo.
Exemplo: desktop_organizator/05-06-23/download.pdf

Neste exemplo, um dos arquivos presentes no diretório organizado era o arquivo-exemplo.txt e sua ultima modificação foi no dia 5 de junho de 2023.

----------

This Ruby script organizes the desktop in the following way:

A new directory is created in the location where the .rb file is, and its name is set as desktop_organizer.

The files present in the specified directory are observed, and new directories are subsequently created with names corresponding to the date of the last edit of each file.
Example: desktop_organizator/05-06-23/download.pdf

In this example, one of the files present in the organized directory was sample-file.txt, and its last modification was on June 5, 2024.
